C1: Recenser et identifier les ressources numériques

Ressources numériques

  • Ressources matérielles
  • Ressources logicielles

Ressources matérielles

Les ressources matérielles sont les composants physiques d'un système informatique.

  • Ordinateurs
  • Serveurs
  • Réseaux
  • Périphériques

Ressources logicielles

Les ressources logicielles sont les programmes et les données qui permettent le fonctionnement d'un système informatique.

  • Systèmes d'exploitation
  • Applications
  • Outils de développement
  • Services en ligne

Recensement ?

Le recensement des ressources numériques consiste à identifier et à documenter toutes les ressources informatiques d'une organisation.

Il est important de recenser les ressources numériques pour:

  • Gérer les actifs informatiques
  • Assurer la sécurité
  • Optimiser les performances
  • Planifier les évolutions

L'identification des ressources numériques permet de:

  • Assurer la traçabilité
  • Faciliter la maintenance
  • Améliorer la gestion des incidents
  • Optimiser les coûts

GLPI

GLPI est un outil de gestion de parc informatique qui permet de recenser et d'identifier les ressources numériques.

Installation de GLPI

GLPI est un programme open-source écrit en PHP qui a besoin d'un serveur web et d'une base de données.

Recensement avec les agents GLPI

GLPI propose des agents qui peuvent être installés sur les machines pour collecter des informations sur les ressources matérielles et logicielles.
